# Smart Apartment Management System 🏢

Yeh ek complete society management system hai jisme Admin, Secretary, Owner aur Security roles hain.

### Tech Stack:
* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
* **Backend:** PHP 8.2
* **Database:** MySQL

### Setup:
1. Sabse pehle `db.php` mein apne database credentials set karein.
2. MySQL mein `smart_building` naam ka database banayein.
3. Tables create karne ke liye SQL file import karein.
